## ソースコードの取得

ワークショップで利用するソースコードは GitHub リポジトリ から取得できます。

```bash
git clone https://github.com/ryokaji/2025-tokai-workshop.git
cd 2025-tokai-workshop
```

## セットアップ

必要なパッケージを取得してください。 pip および uv のいずれでも構いません。

<Tabs>
  <TabItem label="pip">
    <Code
      lang="bash"
      code={`python -m venv .venv --upgrade-deps
source .venv/bin/activate
pip install -e '.'`}
    ></Code>
  </TabItem>
  <TabItem label="uv">
    <Code
      lang="bash"
      code={`uv sync
source .venv/bin/activate`}
    ></Code>
  </TabItem>
</Tabs>

## クライアントの取得

ブラウザで動作する事前ビルドされたクライアントが用意されています。いくつかの方法を提示しますので、いずれかの方法で取得してください。

<Tabs>
  <TabItem label="curl">
    <Code
      lang="bash"
      code={`curl -Ls https://github.com/ryokaji/2025-tokai-workshop/releases/latest/download/1_basic_flow.zip --output 1_basic_flow.zip
curl -Ls https://github.com/ryokaji/2025-tokai-workshop/releases/latest/download/2_user_auth.zip --output 2_user_auth.zip
unzip 1_basic_flow.zip -d frontend/1_basic_flow
unzip 2_user_auth.zip -d frontend/2_user_auth`}
    ></Code>
  </TabItem>
  <TabItem label="wget">
    <Code
      lang="bash"
      code={`wget -q https://github.com/ryokaji/2025-tokai-workshop/releases/latest/download/1_basic_flow.zip
wget -q https://github.com/ryokaji/2025-tokai-workshop/releases/latest/download/2_user_auth.zip
unzip 1_basic_flow.zip -d frontend/1_basic_flow
unzip 2_user_auth.zip -d frontend/2_user_auth`}
    ></Code>
  </TabItem>
  <TabItem label="手動">
    以下の各リンクをクリックして ZIP ファイルをダウンロード、展開してください。
    <CardGrid>
      <LinkCard
        title="1_basic_flow.zip"

    </CardGrid>
  </TabItem>
</Tabs>
